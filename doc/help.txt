Привет , меня зовут Smallik и я гильдейский бот! И я помогу тебе найти интересную для тебя информацию для того что бы узнать:

1) Правила гильдии - введи команду:
```
/info или /инфо
```

2) Системе лута - введи команду:
```
/epgp или /епгп
```

3) Популярные аддоны - введи команду:
```
/addons или /аддоны ( в данный момент на разработке)
```

4) Звания в гильдии и для чего оны нужны  - введи команду:
```
/rank или /звания
```

5) Что бы посмотреть доступные очереди на квесты/ачивы - введи команду:
```
/turn или /очередь ( в данный момент на разработке)
```
